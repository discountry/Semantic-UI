语义UI的代码风格概览。

类名 APIs

声明属性

语义UI使用类名ui来声明页面元素样式。这样有助于区别页面元素的样式class属性和其他类型的class属性。

例如在使用菜单分类UI时，在菜单的页面容器中声明了uiclass属性，而在item中没有声明，因为item包含在菜单分类中。

Home Inbox

改变样式

语义UI使用单个英文单词作为样式的类名。所有的形容词类名都包含在页面元素的类型或者可变属性中。CSS样式定义均为形容词。这样不会破坏语义UI的命名空间。

组合元素

语义UI的所有元素都可以单独或与其他元素组合使用。

通过javascript切换标签。

组件具有一些通用的动作。

$('.demo.menu .item') .tab('change tab', 'second') ;

动作可以组合使用。

对于所有的内部动作都适用。

$('.demo.menu .item') .tab('deactivate all') .tab('activate tab', 'third') .tab('activate navigation', 'third') ;

HTML5 声明

组件可以在任何时候重新初始化。

$('.demo.menu .item') .tab({ history : true, path : '/introduction/getting-started.html' }) ;
