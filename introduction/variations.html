可变属性

语义UI的每类元素都具有一些可以搭配设置的属性。

概览 用法

概览

可变属性可以改变页面元素的样式。多个属性可以搭配使用。

例如，过宽的菜单会占据太多的页面空间，你可以使用 compact 属性来限制菜单的宽度。

独立

在语义UI中，可变属性可以用于描述不同的元素，并产生不同的效果。就像在自然语言中,“大”可以被用来描述一颗大的行星或一只大的昆虫。

例如下面这个例子。

语义UI对于大小的定义均使用了em rem。具有很良好的自适应性和响应式结构。

/* a large form is not quite as large as a large icon */ .ui.large.form { font-size: 1.125em; } /* icons need to adjust vertical alignment as they grow past 1em */ i.large.icon { font-size: 1.5em; vertical-align: middle; }

组合使用可变属性

菜单的另一个属性是inverted menu。它改变了菜单的颜色，和其他颜色熟悉，例如red一同使用会产生不同的样式效果。

可变属性可以组合使用而且不会相互干扰，就像inverted red tiered menu这个例子一样。
